# Matibidi Hospital Website

A modern, responsive website for Matibidi Hospital, showcasing our healthcare services and facilitating patient interactions.

## Features

- **Responsive Design**: Fully responsive website that works on all devices (desktop, tablet, and mobile)
- **Modern UI**: Clean and professional interface with Bootstrap 5
- **Interactive Elements**: Dynamic content and smooth animations
- **Department Information**: Detailed information about all hospital departments
- **Doctor Profiles**: Information about our medical professionals

## Pages

1. **Home Page** (`index.html`)
   - Welcome section with hero image
   - Quick access to key services
   - Emergency contact information

2. **About Us** (`about.html`)
   - Hospital history
   - Mission and vision
   - Core values
   - Facilities information

3. **Services** (`services.html`)
   - Comprehensive list of medical services
   - Service descriptions
   - Emergency services information

4. **Departments** (`departments.html`)
   - Department listings
   - Department-specific information
   - Search functionality
   - Operating hours

5. **Doctors** (`doctors.html`)
   - Doctor profiles
   - Specializations
   - Contact information

6. **Contact** (`contact.html`)
   - Contact form
   - Location information
   - Emergency numbers

## Setup and Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```

2. Navigate to the project directory:
   ```bash
   cd matibidi-hospital
   ```

3. Open `index.html` in your web browser to view the website.
